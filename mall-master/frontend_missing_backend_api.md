# 前端需后端补充的接口清单（按模块分类）

## 1. 品牌模块
- 接口路径+请求方式：/g/v1/brands/:id GET
- 前端现有请求字段：id（数字，用于查询单条品牌详情）
- 前端期望返回字段/内容：brand 详情对象（包含品牌id、名称、logo、描述等）
- 备注：后端文档仅提供品牌列表/增删改接口，无单条详情查询接口

## 2. 品牌分类模块
- 接口路径+请求方式：/g/v1/categorybrands/:id GET
- 前端现有请求字段：id（数字，用于查询单条品牌分类详情）
- 前端期望返回字段/内容：brand/category 详情（包含品牌分类id、品牌id、分类id等）
- 备注：后端文档仅有品牌分类列表接口，无按id查询接口

## 3. 权限模块
- 接口路径+请求方式：/ums/ (GET/POST/PUT/DELETE)
- 前端现有请求字段：roleName、menuIds 等（权限/菜单相关字段）
- 前端期望返回字段/内容：权限列表、菜单树结构等
- 备注：后端文档未覆盖权限模块，无相关接口定义

## 4. 营销模块
- 接口路径+请求方式：/sms/ (优惠券/秒杀/广告等多接口)
- 前端现有请求字段：couponCode、flashId 等（营销相关字段）
- 前端期望返回字段/内容：coupon/flash 相关结构（优惠券信息、秒杀活动信息等）
- 备注：后端文档未提供营销/运营相关接口

## 5. 商品属性&库存模块
- 接口路径+请求方式：/pms/productAttr、/pms/skuStock 等 GET/POST
- 前端现有请求字段：attrName、stock 等（属性/库存字段）
- 前端期望返回字段/内容：属性列表、库存列表（包含属性id、属性名、库存数量等）
- 备注：后端文档未提供商品属性与库存管理接口
